# Registration Form

A simple and responsive Registration Form built using **HTML**, **Tailwind CSS**, and **JavaScript**. The project validates user input in real time and displays a success message after successful registration.

## 🚀 Features

- User-friendly registration form
- Name validation (letters only)
- Email validation (Gmail format)
- Password validation (minimum 6 characters)
- Real-time error messages
- Registration success message
- Responsive UI using Tailwind CSS

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript

## 📂 Project Structure

```
Registration-form/

```

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/ishu122004/Register-form.git
```

2. Open the project folder.

3. Start Tailwind CSS:

```bash
npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css --watch
```

4. Open `input.html` using Live Server.

## 📋 Validation Rules

- **Name:** Only alphabetic characters are allowed.
- **Email:** Must be a valid Gmail address.
- **Password:** Minimum 6 characters.
